Git Work Flow
## update your main branch
```
git checkout main
git push origin main
```
## start work on a feature
```
git checkout -b feature-branch
```

## write code, commit, repeat
```
git add .
git commit
```


## push to a feature branch on YOUR fork
```
git push origin feature-branch
```
